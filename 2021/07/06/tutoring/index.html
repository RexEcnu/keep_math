学习路线

  • 环论 -> 交换代数 -> 代数几何
  • 模论 -> 范畴 -> 同调代数
  • 点集拓扑 -> 代数拓扑 -> 微分流形 -> 李群

数分-高代-解几

  1. 线性代数
    1. 工具层面：矩阵，行列式。多项式
    2. 抽象结构：向量空间，线性变换
    3. 特殊结构：欧式空间，内积空间，二次型
  2. 微积分
    1. 极限思想：推导语言严谨化
    2. 求极限：数列，级数
    3. 连续，微分，积分；一元，多元
  3. 解析几何
    1. 平面方程
    2. 二次曲面

代数课程

  1. 本科
    1. 抽象代数（群，环，域，模）
  2. 研究生
    1. 交换代数（研究交换环）
    2. 代数几何（多项式零点）
    3. 范畴（研究抽象对象的对象）
    4. 同调代数（Abel范畴上的链）
    5. 李群（代数+分析+几何）
    6. hopf 代数，李代数 etc

学习方法和误区

  1. 康奈尔笔记（契合思维习惯，记笔记可用）。
  2. 理论刚接触时，会冒出很多问题，想法，不急着思考，而打断学习。先进行下去，问题想法做个记录，学过一阶段再回头看这些问题。
