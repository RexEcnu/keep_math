唠唠闲话

计算机代数系统(computer algebra system, CAS) 泛指一类支持符号运算的软件，这类软件以类似数学家传统推演的方式，对数学表示式进行操作，常用软件包括 Maxima, Magma, Maple, Mathematica, SageMath, Julia 等。

本篇对这些软件做基本介绍，并附上 Windows 系统的安装教程，Ubuntu 系统的教程移步这里

博客希望尽可能将 CAS 主流软件都做介绍，但是，个人推荐专注其中一两个，在精不在多。特别地，推荐以下三个：

  • SageMath。Sage 语法与 Python 一致，学过 Python 的上手容易（增加了 Lazyness 特性，修改了少量语法规则，比如 ^ 用来代表幂次而不是异或）
  • Julia。Julia 的语言特性对于 CAS 是一大利器。当然，这主要针对开发而言，如果对写工具感兴趣而不是单纯调包，个人首推 Julia。
  • Mathematica。Mathematica 大概处理符号计算的最强工具了，其灵活性和计算性能都很强大，缺点是学习曲线陡，上手有难度。

其他软件，我接触不深，只简单带过。

初学强推 SageMath，其他可以稍作了解。

CAS 软件大致分成两类，一类是专门为代数编程/符号计算而开发设计的，比如 SageMath，GAP， Mathematica；另一类是借助通用语言编写的软件包，比如 Julia 中的 Symbolics.jl 和 OSCAR.jl，Python 中的 Sympy。两种类别没有明确的界限，比如 Mathematica 也可以做图像运算等偏应用的问题，SageMath 某种程度上也可以看成 Python 包。
